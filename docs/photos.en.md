# Reference Photos

[简体中文](photos.zh-CN.md)

These photos are included to help builders identify the physical parts and ports used by this mod. They are reference photos only. Always verify pin labels, polarity, trigger logic, and relay behavior on your own hardware before wiring the brake coils.

## Electronics Bay Overview

![Electronics bay overview](../assets/photos/electronics-bay.jpg)

Example Voron 2.4 electronics bay layout with the Manta M8P, brake relay module, brake power wiring, and four Z brake stepper motors installed.

## Manta M8P Port Reference

![Manta M8P highlighted ports](../assets/photos/manta-m8p-highlighted-ports.png)

The red boxes highlight the Motor3 / M3B stepper connector area used for the `2A` tap and the USB-side GND area used for relay `DC-` in this build.

## Brake Stepper Motors

![Brake stepper motors](../assets/photos/brake-stepper-motors.jpg)

Example 42-series brake stepper motors used for the four Z positions. Keep the normal stepper phase wiring separate from the brake coil wiring.

## Relay Module Reference

![Relay module reference screenshot](../assets/photos/relay-module-reference.jpg)

Relay module purchase/reference screenshot. Treat it as a visual reference only: different relay modules may have different `S`, `DC+`, `DC-`, `COM`, `NO`, and `NC` terminal order, trigger voltage, jumper logic, and contact rating.
