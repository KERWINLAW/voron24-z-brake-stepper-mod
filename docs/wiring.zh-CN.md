# 接线说明

![接线图](../assets/wiring-diagram.zh-CN.png)

## 目标

用继电器控制四个 24V 刹车线圈。继电器控制端由 Manta M8P 的 `M3B-2A` 和 USB 口旁 GND 提供，继电器触点只切换刹车线圈的 24V 正极。

## 端口定义

| 名称 | 说明 |
| --- | --- |
| `M3B-2A` | Manta M8P Motor3 / M3B 端口中的 2A 引脚 |
| USB GND | Manta M8P USB 口旁边的 GND |
| `DC+` | 继电器模块控制侧正极 |
| `DC-` | 继电器模块控制侧负极 |
| `S` / `IN` | 继电器模块信号触发端 |
| `COM` | 继电器受控触点公共端 |
| `NO` | 常开触点，吸合后与 COM 导通 |
| `NC` | 常闭触点，本方案不接 |

## 控制侧接线

1. 从 Manta M8P Motor3 / M3B 的 `2A` 并接一根线。
2. 将这根 `M3B-2A` 线同时接到继电器 `S` 和 `DC+`。
3. 从 Manta M8P USB 口旁 GND 引出一根线。
4. 将 USB GND 接到继电器 `DC-`。
5. 继电器高/低电平触发跳帽按实测调整，使目标状态下继电器吸合并释放刹车。

## 刹车负载侧接线

1. 24V 刹车电源 `V+` 接继电器 `COM`。
2. 继电器 `NO` 接四个刹车线圈的 `+`，四路并联。
3. 四个刹车线圈的 `-` 接 24V 刹车电源 `V- / 0V`。
4. `NC` 不接。

## 不要这样接

- 不要把 24V 刹车电源接到继电器控制端 `DC+ / DC-`。
- 不要让刹车线圈电流经过 Manta M8P 的步进驱动口。
- 不要让刹车线圈电流经过 USB GND。
- 不要切断原来的 Z 步进电机四芯相线。

## 首次测试

1. 不接刹车线圈，只接继电器控制侧，确认继电器能按预期吸合和释放。
2. 接上 24V 刹车电源，但先不要挂载龙门负载，确认 NO 触点输出正确。
3. 接入四个刹车线圈，手托龙门测试释放和抱死。
4. 确认断电、Klipper emergency stop、idle timeout 后刹车会抱死。
