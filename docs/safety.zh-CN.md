# 安全说明

带刹车 Z 步进会影响龙门安全，接线错误可能导致刹车不释放、误释放、继电器损坏、下位机损坏，或龙门掉落。

## 必须确认

- 刹车线圈额定电压是 24V。
- 四个刹车线圈总电流小于继电器触点额定电流，并留有余量。
- 继电器控制端电流不会超过 `M3B-2A` 触发方案的能力。
- USB GND 只用于继电器控制参考地，不承载刹车线圈电流。
- 24V 刹车电源 `V+ / V-` 与继电器控制端 `DC+ / DC-` 不混接。
- 线圈两端有合适的感性负载吸收保护。

## 推荐保护

- 给刹车负载回路加保险。
- 使用合适线径和压接端子。
- 在线圈两端加反向二极管、TVS 或 RC 吸收，具体取决于线圈和继电器模块。
- 首次测试时拆下或支撑龙门负载。
- 测试断电、重启、固件停止、idle timeout 等状态下的刹车行为。
